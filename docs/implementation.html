The interpolation algorithm consists of 4 steps, which are implemented under src/kernel.

  1. Compute vorticity

    \[\omega_z = -\dder{\uy}{x} +\dder{\ux}{y},\]

    which are defined at all cell corners including points on the walls.

  2. Compute scalar potential

    We solve a Poisson equation

    \[\dder{}{x} \left( \dder{\psi}{x} \right) + \dder{}{y} \left( \dder{\psi}{y} \right) = \omega_z,\]

    whose solutions are defined at all cell corners (same locations as \(\omega_z\)). This equation is solved by an FFT-based solver which is adopted in Simple Navier-Stokes Solver.

  3. Interpolate scalar potential

    \(\psi\) is interpolated to the new grid corners. User can choose an arbitrary scheme based on the accuracy, TVD property, etc. In the current example, cubic interpolations are used in \(x\) direction, while information is interpolated in the spectral domain in \(y\) direction thanks to the uniform grid spacing.

  4. Compute velocity

    New velocity field on the interpolated grid is obtained by

    \[\begin{split}\ux &= \dder{\psi}{y}, \\ \uy &= -\dder{\psi}{x},\end{split}\]

    which are of course defined at \(x\) and \(y\) cell faces, respectively. Conventional finite-difference schemes are used.
